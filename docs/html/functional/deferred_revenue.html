> Deferred Revenue¶

  1. สร้าง Business Unit ใน Odoo

  2. Deferred Revenue

  3. Payment Gateway Settlement

  4. Receive Input VAT

สร้าง Business Unit ใน Odoo¶

  1. เข้าไปที่ 945 > Business Unit

  2. กด Create แล้วใส่ข้อมูลของ Business Unit เพื่อสร้าง Business Unit

Deferred Revenue¶

  1. เข้าไปที่ 945 > Operations > Deferred Revenue

  2. กด Create แล้วเลือก Business Unit และใส่จำนวนเงิน

  3. กด Create Move แล้วตรวจสอบข้อมูลในการบันทึกบัญชีให้ถูกต้อง หลังจากนั้นกด Create Journal Entry เพื่อสร้าง Journal Entry

ตัวอย่างการบันทึกบัญชี

Payment Gateway Settlement¶

  1. เข้าไปที่ 945 > Operations > Payment Gateway Settlement

  2. กด Create แล้วเลือก Journal Entry และใส่จำนวนเงินใน Total Amount, MDR, VAT

  3. กด Create Move แล้วตรวจสอบข้อมูลในการบันทึกบัญชีให้ถูกต้อง หลังจากนั้นกด Create Journal Entry เพื่อสร้าง Journal Entry

ตัวอย่างการบันทึกบัญชี

Receive Input VAT¶

  1. เข้าไปที่ 945 > Operations > Payment Gateway Settlement

  2. เลือกรายการที่ต้องการบันทึก Input VAT

  3. ไปที่ Action > Receive Input VAT ระบบจะรวบรายการต่างๆไปไว้ที่ 945 > Operations > Receive Input VAT

  4. เข้าไปที่ 945 > Operations > Receive Input VAT

  5. กด Create Move ที่รายการที่ต้องการ แล้วตรวจสอบข้อมูลในการบันทึกบัญชีให้ถูกต้อง หลังจากนั้นกด Create Journal Entry เพื่อสร้าง Journal Entry

ตัวอย่างการบันทึกบัญชี
